[[solution-strategy]]
= Solution Strategy

The architecture keeps the backend as the system of record, offers web and
mobile clients through an API boundary, and lets the mobile app queue expense
entries while offline.

The first increment is a walking skeleton: create a category, add an expense,
sync it, and show the remaining monthly budget.
