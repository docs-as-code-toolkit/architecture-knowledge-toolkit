= Deployment View

Local development runs backend, web UI, and database with a single compose
setup. The mobile app uses the same backend API endpoint through environment
configuration.

The first hosted environment deploys the backend API and database together in
one region. Static web assets are served separately. Mobile app distribution is
outside the first architecture decision scope.
