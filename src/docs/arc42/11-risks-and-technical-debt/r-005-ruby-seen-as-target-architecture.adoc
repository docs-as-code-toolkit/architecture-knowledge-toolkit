= R-005: Ruby Is Mistaken For A Fixed Target Architecture

== Status

Proposed.

This risk is AI-assisted. It identifies a concern for review and does not
decide risk acceptance.

== Risk Statement

Because the current validator and generator are implemented in Ruby,
contributors may treat Ruby as the accepted long-term implementation language,
leading to premature coupling and reduced openness for future runtime choices.

== Assessment

[cols="1,3", options="header"]
|===
| Field | Value
| Likelihood | Low.
| Impact | Medium.
| Priority | Low.
| Timeframe | Medium-term.
| Confidence | Medium.
|===

== Mitigation Options

* Keep the technology constraints clear that Ruby is the current implementation
  language only.
* Create an ADR before changing or fixing the long-term implementation
  language.
* Use xref:q-dev-001[Q-DEV-001] and xref:q-dev-004[Q-DEV-004] as review
  evidence for future runtime decisions.
