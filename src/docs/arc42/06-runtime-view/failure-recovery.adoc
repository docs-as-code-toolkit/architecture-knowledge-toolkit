== [[failure-recovery]]Failure And Recovery Scenario

[plantuml,arc42-runtime-failure-recovery,svg]
----
@startuml
actor Contributor
participant "Validation and generation scripts" as Scripts
database "Source documentation" as Docs
database "Metamodel schemas" as Schemas

Contributor -> Scripts: Run validation
Scripts -> Docs: Read artifact metadata
Scripts -> Schemas: Validate required fields and relations
Scripts --> Contributor: Fail with actionable message
Contributor -> Docs: Fix missing ID, unknown relation, or broken target
Contributor -> Scripts: Re-run validation
Scripts --> Contributor: Pass
@enduml
----

Recovery rule: generated views must not be accepted when validation fails.

Covered building blocks: validation and generation scripts, source
documentation, metamodel schemas.
