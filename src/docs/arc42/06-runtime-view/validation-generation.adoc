== [[validation-generation]]Validation And Generation Workflow

[plantuml,arc42-runtime-validation-generation,svg]
----
@startuml
actor Contributor
participant "Validation and generation scripts" as Scripts
database "Source documentation" as Docs
database "Metamodel schemas" as Schemas
collections "Generated documentation" as Generated
participant "CI/CD" as CI

Contributor -> Scripts: Run validation locally
Scripts -> Docs: Read AsciiDoc and front matter
Scripts -> Schemas: Load artifact and relation rules
Scripts -> Contributor: Report validation result
Contributor -> Scripts: Generate derived views
Scripts -> Generated: Write deterministic output outside source truth
CI -> Scripts: Repeat validation and generation
Scripts -> CI: Exit with stable result
@enduml
----

Covered building blocks: validation and generation scripts, source
documentation, metamodel schemas.
