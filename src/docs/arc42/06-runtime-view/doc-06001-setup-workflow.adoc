== [[setup-workflow]]Initial Setup Workflow

[plantuml,arc42-runtime-setup,svg]
----
@startuml
actor "Individual architect" as Architect
participant "Source documentation" as Docs
participant "Templates" as Templates
participant "AI skills" as Skills
participant "Human reviewer" as Reviewer

Architect -> Templates: Select setup structure
Architect -> Docs: Create canvases, vision, roadmap, and arc42 skeleton
Architect -> Skills: Request suggested artifact content
Skills -> Docs: Propose draft content
Architect -> Docs: Mark assumptions and open questions
Architect -> Reviewer: Submit source changes for review
Reviewer -> Docs: Accept or request changes
@enduml
----
