|===
| Building block | Responsibility | Interface | Storage location | Affected by

| Source documentation
| Holds reviewed source knowledge and reusable source fragments.
| AsciiDoc files, `include::[]`, anchors, and xrefs.
| `src/docs/`, `docs/`, `examples/`.
| Chapters 1, 2, 3, 4, 8, 10, 11.

| Metamodel schemas
| Define artifact metadata and relation semantics.
| YAML schemas and validator-readable relation definitions.
| `metamodel/`.
| Chapters 2, 4, 8, 9, 10, 11.

| Templates
| Provide repeatable artifact authoring structure.
| AsciiDoc templates copied or adapted by users.
| `templates/`.
| Chapters 1, 4, 8, 9, 10, 11.

| AI skills
| Guide AI-assisted artifact drafting and review.
| Markdown skill instructions and referenced files.
| `skills/`.
| Chapters 2, 4, 8, 9, 10, 11.

| Validation and generation scripts
| Validate source metadata and create deterministic generated views.
| Command-line invocation.
| `scripts/`.
| Chapters 2, 4, 7, 8, 10, 11.

| Adapters
| Isolate engine-specific runtime integration.
| Adapter-specific README and integration notes.
| `adapters/`.
| Chapters 2, 3, 4, 8.
