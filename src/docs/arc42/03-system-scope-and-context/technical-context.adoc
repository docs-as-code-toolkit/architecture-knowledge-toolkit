== [[technical-context]]Technical Context

[cols="1,2,3", options="header"]
|===
| Neighbor | Interface | Notes

| Human reviewer
| Pull request review
| Accepts or rejects source artifacts, metadata, and proposed relationships.

| AI runtime
| Prompt and file-context interaction through tool-specific adapters
| Suggestions must remain marked as suggestions until reviewed.

| Local contributor environment
| Command-line scripts and source files
| Runs validation and generation before review.

| CI/CD
| Validation, generation, and publication jobs
| Reproduces local behavior and publishes derived documentation.

| Documentation publication target
| Static HTML output first
| GitHub Pages is the first automated publication target.

| Consuming repository
| AsciiDoc includes, templates, metadata schemas, and generated fragments
| Adopts only the slices it needs.
|===
