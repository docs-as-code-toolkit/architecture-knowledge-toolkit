== [[error-handling]]Error Handling

[cols="1,3", options="header"]
|===
| Situation | Handling

| Missing required metadata
| Validation fails and reports the missing field and source file.

| Duplicate artifact ID
| Validation fails and reports every conflicting file.

| Unknown relation type
| Validation fails and points to the metamodel-supported relation types.

| Broken relation target
| Validation fails and identifies the missing target ID.

| Generation after validation failure
| No generated documentation is trusted from that run.

| Publication failure
| CI reports the failed publication job; source truth remains unaffected.
|===
