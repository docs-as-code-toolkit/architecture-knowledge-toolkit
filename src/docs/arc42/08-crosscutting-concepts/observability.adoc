== [[observability]]Observability

The toolkit should make automation outcomes inspectable without becoming a
service that requires production telemetry.

[cols="1,3", options="header"]
|===
| Signal | Expected behavior

| Validation output
| Reports missing metadata, duplicate IDs, unknown relation types, and broken
targets with actionable messages.

| Generation output
| Reports created derived files and avoids meaningful diffs when inputs do not
change.

| CI logs
| Show the same validation and generation result that contributors can
reproduce locally.

| Audit trail
| Pull-request history records the human review that turns suggestions into
accepted source.
|===
