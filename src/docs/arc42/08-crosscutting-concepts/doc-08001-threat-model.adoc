== [[threat-model]]Threat Model

[cols="1,1,3,2", options="header"]
|===
| ID | STRIDE | Threat | Affected building block

| T-001
| Tampering
| Generated documentation is edited or committed as source truth.
| Source documentation; validation and generation scripts.

| T-002
| Repudiation
| AI-generated suggestions cannot be distinguished from reviewed content.
| Source documentation; AI skills.

| T-003
| Information disclosure
| Adapter guidance leaks project-specific or environment-specific secrets into
engine-independent skills or dogfood documentation.
| Adapters; AI skills.

| T-004
| Denial of service
| Non-deterministic generation creates unstable diffs and blocks review.
| Validation and generation scripts.

| T-005
| Elevation of privilege
| AI output is treated as accepted architecture authority without human review.
| Source documentation; AI skills.
|===
