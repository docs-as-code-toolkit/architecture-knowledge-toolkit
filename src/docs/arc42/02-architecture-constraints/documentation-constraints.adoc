== [[documentation-constraints]]Documentation Constraints

include::../../fragments/architecture/architecture-constraints.adoc[]

Additional arc42-specific constraints:

* Context, building-block, and runtime chapters contain at least one PlantUML
  diagram.
* Building-block diagrams use C4-PlantUML with `!include <C4/...>`.
* Canvas files are used as input evidence but are not part of the arc42
  documentation set.
* Decisions are documented as ADRs, not only inside
  xref:architecture-decisions[].
* xref:crosscutting-concepts[] follows the required order: threat model,
  security, testing, observability, error handling.
