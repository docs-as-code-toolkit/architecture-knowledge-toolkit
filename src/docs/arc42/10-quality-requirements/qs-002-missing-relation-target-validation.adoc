= QS-002: Missing Relation Target Validation

== Status

Proposed.

The response measure is a reviewable target. Treat uncertain values as
assumptions until confirmed.

== Scenario

[cols="1,3", options="header"]
|===
| Element | Description
| Source | Contributor.
| Stimulus | Adds a relation target that does not exist.
| Environment | Local validation.
| Artifact | Artifact metadata.
| Response | Validation fails with an actionable message.
| Response Measure | Evidence target: command exits non-zero and names the missing target ID.
|===

== Rationale

This scenario concretizes QG-003 Traceability. Stable IDs are useful only when
invalid relation targets are detected before review.
