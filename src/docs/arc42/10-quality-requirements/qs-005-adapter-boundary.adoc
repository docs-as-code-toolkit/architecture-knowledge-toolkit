= QS-005: Runtime-Specific Guidance Stays In Adapters

== Status

Proposed.

The response measure is a reviewable target. Treat uncertain values as
assumptions until confirmed.

== Scenario

[cols="1,3", options="header"]
|===
| Element | Description
| Source | Adapter maintainer.
| Stimulus | Adds runtime-specific invocation guidance.
| Environment | Repository review.
| Artifact | Adapter documentation and skill files.
| Response | Engine-independent skills remain free of runtime assumptions.
| Response Measure | Evidence target: runtime-specific changes are located under `adapters/`.
|===

== Rationale

This scenario concretizes QG-005 Portability. Core skills should stay useful
across engines while adapters carry runtime-specific guidance.
