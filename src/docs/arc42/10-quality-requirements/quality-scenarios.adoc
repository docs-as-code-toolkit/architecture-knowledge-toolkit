== [[quality-scenarios]]Quality Scenarios

[cols="1,1,2,2,2,2,2,2", options="header"]
|===
| ID | Objective | Source | Stimulus | Artifact | Environment | Response | Response measure

| xref:qs-001-reproducible-generation[QS-001]
| QG-001
| Contributor
| Runs the generator twice without changing source inputs.
| Generated documentation.
| Local checkout or CI.
| The generated output remains stable.
| Assumption: second run produces no meaningful diff.

| xref:qs-002-missing-relation-target-validation[QS-002]
| QG-003
| Contributor
| Adds a relation target that does not exist.
| Artifact metadata.
| Local validation.
| Validation fails with an actionable message.
| Evidence target: command exits non-zero and names the missing target ID.

| xref:qs-003-reusable-fragments-render[QS-003]
| QG-004
| Documentation maintainer
| Includes a shared product statement in more than one document.
| Reusable source fragment.
| Plain Asciidoctor-generated static HTML.
| The fragment renders correctly in every include location.
| Assumption: generated HTML contains one rendered copy per include location
and no broken include directive.

| xref:qs-004-ai-suggestion-reviewability[QS-004]
| QG-002
| Architecture reviewer
| Reviews an AI-suggested architecture artifact.
| Source artifact and metadata.
| Pull request review.
| The reviewer can distinguish suggested content from accepted source truth.
| Assumption: status or prose marker is visible in the reviewed artifact.

| xref:qs-005-adapter-boundary[QS-005]
| QG-005
| Adapter maintainer
| Adds runtime-specific invocation guidance.
| Adapter documentation and skill files.
| Repository review.
| Engine-independent skills remain free of runtime assumptions.
| Evidence target: runtime-specific changes are located under `adapters/`.
|===

confirmation before these scenarios can become final.
