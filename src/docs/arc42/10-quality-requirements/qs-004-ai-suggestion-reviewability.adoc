== Status

Proposed.

The response measure is a reviewable target. Treat uncertain values as

== Scenario

[cols="1,3", options="header"]
|===
| Element | Description
| Source | Architecture reviewer.
| Stimulus | Reviews an AI-suggested architecture artifact.
| Environment | Pull request review.
| Artifact | Source artifact and metadata.
| Response | The reviewer can distinguish suggested content from accepted source truth.
| Response Measure | Assumption: status or prose marker is visible in the reviewed artifact.
|===

== Rationale

This scenario concretizes QG-002 Reviewability. The repository permits
AI-assisted drafting, but accepted architecture truth must remain a human
review decision.

== Assumptions

* Proposed or draft metadata is visible in review.
* Human acceptance is recorded before an artifact becomes accepted.
