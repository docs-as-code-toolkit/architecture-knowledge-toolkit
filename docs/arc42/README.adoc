This directory is reserved for arc42-based architecture documentation.

Use this area for structured architecture documentation such as:

* Introduction and goals.
* Constraints.
* Context and scope.
* Solution strategy.
* Building block view.
* Runtime view.
* Deployment view.
* Cross-cutting concepts.
* Architecture decisions.
* Quality requirements.
* Risks and technical debt.

Documents should use AsciiDoc and include stable metadata in the document header
or in an explicit sidecar metadata file.
