# Talks

This directory collects talk-specific material for events, meetups, and
conference proposals.

Each talk gets its own subdirectory so slides, abstracts, demo scripts,
speaker notes, and follow-up material can evolve independently per event.

## License

By default, talks in this directory are covered by the repository's
[MIT License](../LICENSE), like the rest of the project.

An author may place their talk under stricter terms by adding a `LICENSE` file
to that talk's own subdirectory. Such a per-talk `LICENSE` applies only to that
subdirectory and overrides the default for it. When contributing a talk, either
leave it under the MIT default or include your own `LICENSE` in your talk's
folder.

Example: [`ai-assisted-coding-meetup-envite/`](ai-assisted-coding-meetup-envite/)
is licensed under CC BY-NC-ND 4.0 — see its
[LICENSE](ai-assisted-coding-meetup-envite/LICENSE).
